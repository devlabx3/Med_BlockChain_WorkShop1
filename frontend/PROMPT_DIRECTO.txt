-----------

   - Comida aparece en posición aleatoria
   - Game Over cuando choca PARED o CUERPO PROPIO
   - Score: +10 puntos por comida comida
   - Serpiente crece al comer (no pop tail)

     * updateScore(newScore) - guardar score (solo si > topScore actual)
     * getPlayer(address) - leer nickname y topScore
     * playerExists(address) - verificar si está registrado
   - RPC: https://monad-testnet.g.alchemy.com/v2/uAf5-PCiQhSYESR3WOt81
   - WalletConnect ID: 05b0564c6afd9cca5c24433117d22f43

   - TanStack React Query v5
   - CSS Grid + Gradientes + Animaciones

4. UI COMPONENTS
   - GameBoard: Renderiza 20x20 grid con snake y food
   - PlayerPanel: Panel de user (wallet, registro, save score)

5. DISEÑO
   - Tema espacial: fondo oscuro (#000814), estrellas animadas
   - Colores neón: verde primario (#00ff88), azul secondary (#0099ff)

REGLAS CRÍTICAS (RainbowKit):
----------------------------

1. config.js DEBE tener:
   - import { monadTestnet } from "wagmi/chains" (NUNCA hardcodear chain)
   - export const wagmiConfig = getDefaultConfig({ ... ssr: false ... })
   - export const CULEBRITA_ABI = [...]
   - export const CULEBRITA_CONTRACT_ADDRESS = import.meta.env.VITE_...

2. main.jsx DEBE tener (ORDEN FIJO):
   - import "@rainbow-me/rainbowkit/styles.css" (una sola vez)
   - WagmiProvider → QueryClientProvider → RainbowKitProvider (este orden)
   - QueryClient({})

3. .env DEBE tener:
   VITE_WALLETCONNECT_PROJECT_ID=05b0564c6afd9cca5c24433117d22f43
   VITE_CHAIN_RPC_URL=https://monad-testnet.g.alchemy.com/v2/uAf5-PCiQhSYESR3WOt81
   VITE_CULEBRITA_CONTRACT_ADDRESS=0xF6F99CE3FFe2747EdAF79402a7F48414340A3a5C

4. App.jsx DEBE tener:
   - useAccount() para obtener address
   - useReadContract() para getPlayer y playerExists
   - useWriteContract() para registerPlayer y updateScore
   - useWaitForTransactionReceipt() para ambas writes
   - useEffect(() => { if (isSuccess) refetch(); }, [isSuccess])
   - Button disabled={isPending || isConfirming}

5. Game Loop:
   - setInterval ~200ms
   - Cálculo newHead
   - CHECK: pared (x<0 || x>=20 || y<0 || y>=20) → GAME OVER
   - CHECK: cuerpo propio → GAME OVER
   - CHECK: comida → +10 score, crecer, nueva comida
   - Mover snake

6. PlayerPanel DEBE:
   - Leer playerExists(address)
   - Si no existe: form register con input nickname
   - Si existe: mostrar nickname, topScore, currentScore
   - Si currentScore > topScore: botón "Guardar Score"
   - Guardar en localStorage también para leaderboard

7. Leaderboard DEBE:
   - Leer localStorage "culebrita-leaderboard"
   - Mostrar top 10 ordenado por score DESC
   - Escuchar evento "leaderboard-updated" para refrescar

MECÁNICA DEL JUEGO:
-------------------
- Tablero: 20x20
- Snake: comienza en [10,10]
- Food: random position
- Movimiento: 200ms interval
- Colisiones:
  * Pared → Game Over
  * Cuerpo → Game Over
  * Comida → +10 score, crecer
- Crecimiento: no pop tail cuando come

3. Jugar y acumular score
